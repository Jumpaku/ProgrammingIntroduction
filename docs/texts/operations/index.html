演算の優先順位

複数の演算子を同時に利用する場合，優先順位の高い演算子から計算されます． 以下は演算子の優先順位です．

  1. !
  2. *, /, %
  3. +, -
  4. <, >, <=, >=
  5. ==, !=
  6. &&
  7. ||

また， (, ) で囲まれた部分は優先されるため， 以下のように演算の優先順位を指定することができます．

(2 + 7) * (3 - 5) // -18

まとめ

  1. 演算とは演算子を用いた基本的な計算のことである．
  2. 演算には数値の四則演算の他に文字列の結合，大小比較，論理演算などがある．
  3. 演算子には優先順位があるが，(, ) が優先される．
